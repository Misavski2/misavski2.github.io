Sobre

Sou um entusiasta de tecnologia e estudante de desenvolvimento backend, fascinado pela arquitetura e pela lógica que sustentam o mundo digital. Minha jornada na programação é movida pela curiosidade de entender como sistemas complexos funcionam e pela satisfação de construir algo do zero.
Atualmente, dedico meus estudos a entender como criar APIs seguras e modelar bancos de dados eficientes, aplicando esses conceitos em projetos pessoais e acadêmicos. A sensação de desvendar um problema de lógica desafiador é o que me impulsiona a aprender cada vez mais.

Quando não estou codificando, você provavelmente vai me encontrar estudando um novo idioma ou explorando gêneros musicais. Acredito que essa busca por diferentes formas de conhecimento me torna um solucionador de problemas mais versátil e criativo.
Estou buscando ativamente minha primeira oportunidade de estágio ou uma posição júnior, onde eu possa aplicar meu conhecimento, aprender com profissionais experientes e contribuir para a construção de soluções reais. Sou um aprendiz rápido, dedicado e estou pronto para transformar minha paixão em uma carreira de sucesso.

Tecnologias e Ferramentas

C#

Dedico meus estudos em C# para construir uma base sólida em lógica de programação e nos princípios da Orientação a Objetos. Atualmente, aplico esses conceitos no desenvolvimento de aplicações de console para resolver problemas e aprimorar minhas habilidades. E também com o foco em me tornar um .NET Developer.

SQL

Estou aprendendo a linguagem SQL para estruturar informações e realizar consultas em bancos de dados relacionais. Pratico a criação de queries para selecionar, inserir, atualizar e deletar dados, entendendo como a informação é organizada e recuperada.

JavaScript

Utilizo JavaScript para dar vida às minhas páginas, manipulando o DOM para criar interações básicas e responder a eventos do usuário. É a ferramenta que estou usando para transformar páginas estáticas em experiências mais dinâmicas e funcionais.

HTML e CSS

Construo a estrutura de projetos web (inclusive este!) com HTML semântico e os estilizo com CSS moderno, também me aventuro e aprendo muito utilizando frameworks como Tailwind e Bootstrap. É a base visual de todo o meu aprendizado em desenvolvimento web.
